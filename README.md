# 音灵AI SoulMelody 🎵

AI驱动的歌曲生成平台，让每个人都能创作属于自己的音乐。

## 项目简介

音灵AI是一个专注于AI歌曲生成的平台，支持：
- 🎤 歌词生成与优化
- 🎼 旋律创作
- 🎹 人声合成
- 🎧 完整歌曲生成

## 技术架构

```
┌─────────────────────────────────────────┐
│              API Gateway                │
│           (Fastify + REST)              │
├─────────────────────────────────────────┤
│              Core Engine                │
│      (歌曲生成/音频处理/AI模型)          │
├─────────────────────────────────────────┤
│              Storage Layer              │
│      (本地文件存储 + 元数据管理)          │
└─────────────────────────────────────────┘
```

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 运行
pnpm start
```

## 项目结构

```
src/
├── api/           # API 路由与控制器
├── core/          # 核心业务逻辑
├── storage/       # 存储层实现
└── utils/         # 工具函数
```

## 存储方案

当前阶段使用本地文件存储：
- 音频文件：`/data/audio/{userId}/{songId}/`
- 元数据：`/data/meta/{userId}.json`
- 临时文件：`/tmp/soulmelody/`
